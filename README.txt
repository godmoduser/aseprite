Как подключить группу к программе:

1) Создаём группу

2) В разделе "Работа с api" в настройках группы получаем токен, лучше дать ему все возможности.

3) Там же в подразделе longpoll api включаем longpoll, выставляем версию api 5.80

4) Во вкладке "Типы событий" на странице longpoll api находим пункт "Входящие сообщения" и ставим галочку

5) Копируем токен групы на новую строчку в файл "tokens.txt" и числовое id группы (целое число) в файл "ids.txt"

6) Во вкладке "Сообщения" настроек группы включаем сообщения, в подразделе "Настройки для бота" разрешаем добавлять сообщество в беседы.
